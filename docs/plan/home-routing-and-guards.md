# 首页路由模式与连接中操作限制

## 路由模式（首页）

- 仅支持 **规则分流**（`rule`）与 **全局代理**（`global`）。
- 历史 store 值 `direct` 在读取时迁移为 `rule`。
- 连接中切换路由或 TUN 模式会提示先断开（与切换订阅一致）。

## 连接中禁止的订阅操作

- 切换活跃订阅、更新订阅、删除订阅、批量更新：均需引擎空闲。

## 网络设置

- **绕过列表** 写入 `settings.json` 的 `proxy_bypass_key`，由 `sysproxy` 在设置系统代理时读取。
- 修改混合端口、控制台端口或绕过列表后，若引擎在运行，提示断开后重连生效。
